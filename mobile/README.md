# Atarashii App Mobile

Base inicial do app real em Expo/React Native, planejada para Android e iOS.

## Objetivo

Reaproveitar a diretriz, os dados JSON, os assets e os fluxos validados no prototipo web, criando uma base mobile-first para o MVP.

## Como Rodar

Instale as dependencias:

```bash
npm install
```

Inicie o Expo:

```bash
npm start
```

Depois, use:

- Android: `npm run android`
- iOS: `npm run ios`
- Expo Go: escaneie o QR Code do terminal

## Estrutura

- `App.js`: primeira versao funcional do app mobile.
- `src/data/`: dados JSON do MVP.
- `src/assets/`: logo, imagens de bases, katas e regras.

## Observacoes

- O prototipo web continua em `prototype/` apenas como referencia de UX.
- O app mobile e o caminho recomendado para o MVP real.
- Campos futuros de video ja existem em Katas e Kihon, mas podem ficar vazios no MVP.
